# opengenus

CLI tool for finding webpage size and &lt;a> tag count

## Usage

```bash
$ node src/index.js <URL>

# example

$ node src/index.js https://facebook.com
```

### Dependencies

- [jsdom](https://github.com/jsdom/jsdom)
